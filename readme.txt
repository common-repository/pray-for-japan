Pray for Japan icon on your side bar.

== Description ==

This simple plugin add 'Pray for Japan' designed icon on your side bar.

== Installation ==

Place pray-for-japan folder in your /wp-content/plugins/ directory
and activate through the administration panel, and then go to the widget panel and
drag it to where you would like to have it!
